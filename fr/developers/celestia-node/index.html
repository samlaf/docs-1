Node Celestia

Ce tutoriel passe en revue la construction et l'installation d'un nœud Celestia. Ce tutoriel suppose que vous avez terminé les étapes de configuration de votre propre environnement de développement ici.

Installer un node Celestia​

  • Arabica
  • Mamaki
  • Ubuntu (AMD)
  • Ubuntu (ARM)
  • Mac (Apple)
  • Mac (Intel)

Installer celestia-node sur le devnet Arabica signifie en installer une version compatible spécifique à ce réseau.

Install the celestia-node binary by running the following commands:

cd $HOME
rm -rf celestia-node
git clone https://github.com/celestiaorg/celestia-node.git
cd celestia-node/
git checkout tags/v0.5.0-rc5
make install
make cel-key

Verify that the binary is working and check the version with the celestia version command:

celestia version
Semantic version: v0.5.0-rc5
Commit: 039facd94104c741f8fa995e80f1321776ae4ad3
Build Date: Tue Nov 15 16:23:30 EST 2022
System version: amd64/linux
Golang version: go1.19.1

Sélection du réseau​

  • Arabica
  • Mamaki

Vous pouvez choisir le réseau de celestia-node entre Arabica et Mamaki. Cependant, vous noterez que les réseaux fonctionnent mieux sur les versions de celestia-node mentionnées ci-dessus.

celestia light init --p2p.network arabica-2
celestia light start --p2p.network arabica-2

NOTE: It is advised before switching networks to reinitialize your node via init command. Ceci est dû à la présence d'une ancienne configuration. La réinitialisation réinitialisera la configuration.

Errors​

Si vous rencontrez une erreur, c'est probablement parce qu'une ancienne configuration est présente:

Error: nodebuilder/share: interval must be positive; nodebuilder/core: invalid IP addr given:

# ou

Error: nodebuilder/share: interval must be positive

You can re-initialize your node with the following command:

rm -rf ~./celestia-light-arabica-1 && celestia light init

Votre sortie devrait être simialaire à:

@ ~ % rm -rf ./celestia-light-arabica-1 && celestia light init
2022-11-01T22:23:19.581+0100    INFO    node    nodebuilder/init.go:20  Initializing Light Node Store over '/Users/joshstein/.celestia-light-arabica-1'
2022-11-01T22:23:19.582+0100    INFO    node    nodebuilder/init.go:51  Saving config   {"path": "/Users/joshstein/.celestia-light-arabica-1/config.toml"}
2022-11-01T22:23:19.582+0100    INFO    node    nodebuilder/init.go:52  Node Store initialized

Ensuite, démarrez votre node:

celestia light start --node.network arabica-1

La sélection de réseau sur la v0.3.0-rc2 n'est pas disponible.
